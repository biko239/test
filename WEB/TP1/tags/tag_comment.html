• Il est utile de commenter un document lorsqu'il depasse un certain nombre de lignes de code
  • C'est possible en utilisant les balises suivantes:
  • <!--permet de definir un commentaire
  • --> permet de clore un commentaire
    • Le texte qui est entre ces deux balises,n'apparaitra pas dans le document HTML
Exemple:

<!--un commentaire-->
